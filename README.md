# NumScan
![Logo](Photos/Logo.jpg)
NumScan is a simple, fast and practical tool to obtain information from any phone number
# Info
Works on:
 
 **[*] Termux ✓**
 
 **[*] Kali Linux ✓**
 
 **[*] Kali Nethunter ✓**
 
 **[*] Any other distro ✓**
 
 **[*] Windows (CMD) ❌**

_______________________

Information you will get with a phone number:

- Nacional format
- International format
- country
- region
- see if it's valid
- carrier
- time zone
